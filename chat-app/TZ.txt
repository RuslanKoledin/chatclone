ТЗ: UI/UX корпоративного мессенджера (Modern Chat UI)

=== 1. Общая концепция ===
- Интерфейс в стиле: Slack / WhatsApp Web / Telegram Desktop
- Минималистичный, современный, с акцентом на удобство
- Основной цвет: зелёный (accent)
- Базовые цвета: тёмный sidebar + светлая рабочая область

=== 2. Общая структура интерфейса ===
Приложение состоит из 3 основных колонок:

--- 2.1 Левая панель (Sidebar) ---
Назначение: навигация, список разделов, быстрый доступ к чатам
Содержимое:
  [x] Лого / название (HC - Helper Chat)
  [x] Кнопка "+ New Chat"
  [x] Меню:
      [x] Chats
      [x] Groups
      [x] Contacts
      [ ] Calls
      [x] Settings
  [ ] Блок избранных чатов (Favorites)
  [ ] Список недавних чатов (Recent)
  [x] Информация о текущем пользователе (внизу)
Поведение:
  [x] Hover → подсветка
  [x] Active → выделение (фон + индикатор слева)
  [x] Скролл при переполнении

--- 2.2 Центральная панель (Chat List) ---
Назначение: список всех чатов пользователя
Каждый элемент чата содержит:
  [x] Аватар
  [x] Название чата / имя пользователя
  [x] Последнее сообщение
  [x] Время последнего сообщения
  [x] Индикатор непрочитанных сообщений (badge)
  [x] Онлайн статус (зелёная точка)
Поведение:
  [x] Hover → подсветка
  [x] Active чат → выделяется
  [x] Клик → открывает чат справа

--- 2.3 Правая панель (Chat Window) ---
Назначение: отображение переписки

2.3.1 Header (верхняя панель):
  [x] Название чата
  [x] Количество участников (для группы)
  [x] Онлайн статус
  [x] Кнопки:
      [x] поиск 🔍
      [ ] звонок 📞
      [x] информация ℹ️ (меню)

2.3.2 Область сообщений:
  [x] Вертикальный скролл
  [x] Ограничение ширины сообщений (~60%)
  [x] Разделение сообщений по сторонам:
      [x] свои → справа (зелёные)
      [x] чужие → слева (белые)

Формат входящего сообщения:
  [x] Аватар слева
  [x] Имя отправителя (для групп)
  [x] Текст сообщения
  [x] Время
  [x] Фон: белый
  [x] Тень (box-shadow)

Формат исходящего сообщения:
  [x] Выравнивание справа
  [x] Фон: зелёный (#10B981)
  [x] Белый текст
  [x] Время
  [x] Статус: ✔ отправлено / ✔✔ доставлено

Дополнительно:
  📎 Файлы:
  [x] Отображаются как карточка (иконка, название, размер, кнопка скачать)

  🎤 Голосовые сообщения (опционально):
  [x] кнопка play (audio player), waveform (нет), длительность (нет)

  💬 Reply (ответ):
  [x] отображается сверху сообщения (имя, кусок текста, вертикальная линия слева)

2.3.3 Поле ввода сообщения:
  [x] кнопка эмодзи 😊
  [x] кнопка прикрепления 📎
  [x] поле ввода (multiline)
  [x] кнопка отправки (иконка самолётика)
Поведение:
  [x] Enter → отправка
  [x] Shift+Enter → перенос строки
  [x] автоподстройка высоты поля (до 5 строк)

=== 3. Цветовая схема ===
  [x] --sidebar-bg: #111827
  [x] --chat-bg: #F9FAFB

=== 4. UI детали ===
Скругления:
  [x] сообщения: 16px
  [x] кнопки: 12–20px
Тени:
  [x] лёгкие (box-shadow: 0 2px 8px rgba(0,0,0,0.05))
Отступы:
  [x] 8px / 16px / 24px (единая система)

=== 5. Анимации ===
Обязательные:
  [x] появление сообщений (fade + slide)
  [x] hover эффекты
  [x] кнопки (scale при нажатии)

=== 6. Real-time ===
Использовать: WebSocket
Функции:
  [x] мгновенное появление сообщений
  [x] индикатор "печатает..."
  [x] обновление статуса онлайн

=== 7. Технологии ===
Frontend: React + TypeScript, Material UI ✔
State: Redux ✔
Backend: Spring Boot + WebSocket ✔

=== 8. API (минимум) ===
Сообщения: отправка ✔, получение ✔, история ✔
Пользователи: список ✔, статус онлайн ✔
Чаты: создание ✔, список ✔

=== 9. UX требования ===
  [x] пользователь не может написать, пока не выбрал чат
  [x] сообщения не теряются при перезагрузке
  [x] чат автоматически скроллится вниз

=== 10. Критерии приёмки ===
  [x] чат открывается
  [x] сообщения отправляются
  [x] сообщения приходят в real-time
  [x] UI соответствует макету
  [x] есть разделение входящих/исходящих
  [x] есть список чатов

=== 11. Компонентная структура ===
  [x] Sidebar (навигация — тёмная левая панель)
  [x] ChatList (центральная панель — белая)
  [x] ChatWindow (правая панель — MessagePage)
  [x] Message (MessageCard)
  [x] InputBar (встроен в MessagePage, multiline)
